# Angular Resource from https://t.me/AngularWeb

## Clone the repo

```shell
git clone https://github.com/angulartelegram/angular7.git
cd angular7
```

### Install npm packages

Install the `npm` packages described in the `package.json` and verify that it works:

```shell
npm install
ng serve -o
```
